Hardware Setup
==============

.. tab-set::

   .. tab-item:: English

      For this project, no external hardware is required besides the Arduino board itself and a USB cable.

      **Requirements:**
      - Arduino Nano ESP32 (or any other compatible Arduino board).
      - USB-C Cable to connect to the computer.
      - Built-in LED on the board.

   .. tab-item:: Português

      Para este projeto, não é necessário hardware externo além da própria placa Arduino e de um cabo USB.

      **Requisitos:**
      - Arduino Nano ESP32 (ou qualquer outra placa Arduino compatível).
      - Cabo USB-C para ligação ao computador.
      - LED integrado na placa.
