Smart Home Control with Arduino Nano ESP32
==========================================

.. tab-set::

   .. tab-item:: English

      Welcome to the documentation for the **Smart Home Control** project! 
      This project explores the development of a centralized web interface for monitoring and controlling IoT nodes (specifically the Arduino Nano ESP32) using the MQTT protocol.

      The focus is on security, performance, and a premium UI/UX with an industrial aesthetic.

      .. note::
         This project was developed as a learning journey, transitioning from simple embedded temperature monitoring to a full-stack IoT solution.

   .. tab-item:: Português

      Bem-vindo à documentação do projeto **Smart Home Control**!
      Este projeto explora o desenvolvimento de uma interface web centralizada para monitorização e controlo de nodos IoT (especificamente o Arduino Nano ESP32) utilizando o protocolo MQTT.

      O foco principal é a segurança, performance e uma experiência de utilizador (UI/UX) premium com estética industrial.

      .. note::
         Este projeto foi desenvolvido como uma jornada de aprendizagem, passando de uma simples monitorização de temperatura para uma solução IoT completa.

Background
----------

.. tab-set::

   .. tab-item:: English

      This project started with a simple goal: learning how to use an NTC 10Kohm thermistor with an Arduino Nano ESP32. What began as a basic temperature sensor evolved into a comprehensive Smart Home system.

   .. tab-item:: Português

      Este projeto começou com um objetivo simples: aprender a usar um termistor NTC 10Kohm com um Arduino Nano ESP32. O que começou como um sensor de temperatura básico evoluiu para um sistema de Smart Home abrangente.
